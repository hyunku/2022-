# 최적 입지 선정 개요

1. 최적 입지 후보지 선정 (MCLP + P-Median)
2. 구별 물가 현황 파악하여 최적 입지 후보지 추리기
3. 동별 전월세 데이터로 땅값을 고려하여 최적 입지 후보지 추리기
4. 추려진 입지로 자립준비청년의 독립을 위한 주거시설을 설치.
